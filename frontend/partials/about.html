About this web app

This web app was built using Jade (or jspm), Angular, Coffee Script, KarmaJS, SystemJS. Not MEAN stack but JACKS stack.

The basic premise of the website is to collate all the river and sea flood data from the Environment Agency, creating a resource which can be used as part of your property buying research. Unbelievably, very little of this type of data is as easy to access.

The code

You can see the source code on github. The app is hosted in the United States so you may find it a little slow to load depending on your location.

The app has been a project with which I can practice and develop skills. I am interested in Javascript module loaders at the moment, and am getting into DevOps for front-end developers.
